## Usage

Simply open the `index.html` file in any modern web browser to view the website.

To customize the website:

- **Update Content**: Edit the text directly within the `index.html` file.
- **Change Images**: Replace `kwami-happy-hour-headshot-philadelphia-48439-1.png` and other `src` attributes for project images with your own.
- **Update Links**: Change `https://calendly.com/your-calendar-link` to your actual Calendly link, and update social media and project links in the footer and project cards.
- **Google Analytics**: Replace `G-XXXXXXXXXX` with your Google Analytics Measurement ID in the `<head>` section.
- **Favicon**: The current favicon is an emoji. You can replace it with a custom `.ico` or `.png` file by updating the `<link rel="icon" ...>` tag.
